# 🧪 ChemiCraft

**ChemiCraft** é uma aplicação interativa para o ensino de química orgânica, permitindo que usuários montem moléculas de forma visual e intuitiva.  
Atualmente em fase **MVP**, o projeto busca explorar mecânicas lúdicas para o aprendizado de química, com possibilidades futuras de mini games, validações automáticas e geração de nomenclaturas químicas.

## 🚀 Tecnologias Utilizadas

### Frontend
- [Next.js](https://nextjs.org/) — Framework React para renderização rápida e otimizada.
- Canvas API — Manipulação de átomos e estruturas químicas de forma visual.
- [Capacitor](https://capacitorjs.com/) — Planejado para levar o projeto ao mobile.

### Backend (Planejado)
- [NestJS](https://nestjs.com/) — Para validações químicas avançadas, nomenclatura e lógica de regras.

### Ferramentas
- [GitHub Copilot](https://github.com/features/copilot) — Utilizado como agente para acelerar o desenvolvimento e auxiliar em implementações.

## 📦 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/AntDavi/ChemiCraft.git
```

2. Acesse o diretório do projeto:
```bash
cd ChemiCraft
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

5. Acesse no navegador:
```
http://localhost:3000
```

## 🌟 Funcionalidades Atuais
- Barra de átomos para arrastar e soltar no canvas.
- Montagem visual de moléculas.
- Botão para limpar o canvas.
- Exibição do nome químico básico da molécula.

## 🛠 Funcionalidades Futuras
- Validações químicas mais rigorosas.
- Geração automática de nomenclaturas IUPAC.
- Mini games educativos.
- Versão mobile com Capacitor.
- Integração com backend NestJS.
